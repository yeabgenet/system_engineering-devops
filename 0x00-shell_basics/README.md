# 0x00-shell_basics

Shell basics exercises – printing cwd, listing files, changing directory, etc.

## Scripts

| File | Purpose |
|------|---------|
| `0-current_working_directory` | Prints the absolute path of the current working directory (`pwd`). |
| `1-listit` | Lists the contents of the current directory (`ls`). |
| `2-bring_me_home` | Changes the working directory to the user’s home directory (`cd ~`). |
| `3-listfiles` | Lists the current directory contents in long format (`ls -l`). |
| `4-listmorefiles` | Lists **all** contents (including hidden files) in long format (`ls -la`). |

All scripts start with `#!/bin/bash` and are executable.
